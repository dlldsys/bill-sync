# UI 主题更新日志

## 更新日期
2026-03-26

## 更新内容

### 1. 全局样式主题更新
**文件**: `src/styles/global.css`

#### 颜色系统
- **主色调**: 从紫色渐变 (#667eea → #764ba2) 改为 **青色渐变** (#0abfca → #08a9b5)
- **背景色**: 浅灰蓝色调 (#f0f4f8, #ecf0f3)
- **卡片背景**: 纯白色 (#ffffff)
- **深色主题**: 支持深蓝黑色调 (#10151e, #202a37)

#### 设计系统
- **圆角**: 统一使用柔和圆角 (6px / 10px / 14px)
- **阴影**: 轻柔的阴影效果
- **间距**: 标准化间距系统 (4px / 8px / 12px / 16px / 20px)

### 2. 按钮样式
- 主按钮: 青色渐变背景 + 阴影效果
- 悬停效果: 向上位移 + 阴影增强
- 激活效果: 轻微缩放

### 3. 卡片样式
- 圆角: 14px
- 背景: 白色
- 阴影: 中等阴影
- 边框: 1px 透明边框增加层次感

### 4. 账单卡片
- 更大的图标尺寸 (48px)
- 悬停效果: 向右位移
- 金额加粗显示

### 5. 表单元素
- 圆角: 10px
- 聚焦效果: 青色边框 + 光晕
- 下拉框: 自定义箭头图标

### 6. 分类选择
- 图标背景: 青色半透明
- 选中状态: 青色背景 + 白色图标

### 7. 底部导航
- 固定定位
- 顶部阴影
- 选中图标: 缩放效果
- 颜色: 非选中灰色，选中青色

### 8. 页面头部
- 青色渐变背景
- 更大的底部内边距 (50px)
- 柔和阴影
